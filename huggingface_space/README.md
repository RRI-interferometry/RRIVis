# RRIVis - Radio Interferometer Visibility Simulator

Interactive demo of [RRIVis](https://github.com/kartikmandar/RRIvis), a GPU-accelerated visibility simulator for radio interferometry using the RIME (Radio Interferometer Measurement Equation).

## Features

- Configure antenna arrays with preset layouts or custom positions
- Set observation parameters (frequency, time, location)
- Simulate visibilities with full polarization support
- Visualize antenna layouts, UV coverage, and visibility spectra

## Usage

1. Configure your simulation parameters in the sidebar
2. Click "Run Simulation" to compute visibilities
3. Explore the results in the interactive plots
